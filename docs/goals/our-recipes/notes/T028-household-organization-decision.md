# T028 — Household organization decision

## Decision

Select global tag governance and profile settings. They are direct, safe gaps in the present household workflow: profiles remain a non-auth attribution layer, while tags currently exist only as anonymous recipe strings.

## Ranking

1. **T029 (selected):** profile edit/archive/active controls and global tag colors, usage, rename, merge, delete, and editor autocomplete.
2. **Collections:** build on the same household-managed organization semantics after T029.
3. **Document/OCR import:** requires an explicit local file, parser, review, and recovery boundary before it can accept PDFs or handwriting.
4. **AI provider:** requires deterministic mock behavior and credential-safe configuration before any optional live request.
5. **Operational/final evidence:** Docker daemon/Unraid host proof, print/performance breadth, and final acceptance audit wait for feature packages.

## Boundary

T029 must preserve every recipe association during tag changes, prevent archiving the active/last usable profile without a safe replacement, and never imply that profiles are access control.
